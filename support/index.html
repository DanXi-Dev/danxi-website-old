FAQ

  • Q：旦夕安全吗？

    A：旦夕基于 GPL-3.0 License 协议在 GitHub 开源，项目的所有代码都可以查看。这意味着，旦夕可以接受所有人的审查，不会包含任何恶意代码。

  • Q：我的 UIS 账号安全吗？

    A：你的学号和密码只会保存在本地，不会上传至旦夕的服务器。

  • Q：我的个人数据安全吗？

    A：你的个人数据是通过你保存的 UIS 账号直接从复旦大学服务器获取并在旦夕里显示的。你的任何个人数据都不会被上传至旦夕的服务器。

  • Q：遇到了 BUG 怎么办？

    A：请给我们发送邮件，或直接前往 GitHub 提交 Issue。请注意，你的反馈中应该包含

    1. 设备系统版本
    2. 设备型号
    3. 旦夕 App 版本号
    4. 问题及复现手段
    5. 其他补充描述

    如：

    1. iOS 14.5.1
    2. iPhone 12
    3. 0.3.4
    4. 打开就崩溃

  • Q：想要提前获取新功能！可以让我提前获取早期发布的版本吗？

    A：感谢支持！请通过邮件联系我们。
